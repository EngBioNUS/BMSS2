How to use:

Use the simplesbml.py file here instead of the default file in the sitepackage as this includes addtional units not found in the original.


To convert the an existing model in the database into a SBML file:
Run the DatabasetoSBML file. Instructions will be within the console. SBML files will be outputed into this folder.

To add and .ini file to the database and then convert it into SBML
Use the FiletoDatabasetoSBML file. Change the path within the script to match where the .ini files are located and hit run. SBML files will be outputed into this folder.
